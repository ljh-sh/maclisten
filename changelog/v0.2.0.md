## Add

- `maclisten watch` — continuously listen for voice keywords/commands.
  - Restarts recognition sessions automatically to avoid framework time limits.
  - Supports `--keyword` to filter emitted segments.
  - Supports `--partial` to stream intermediate results as JSON lines.
- `--output <path>` for `mic` and `watch` — save microphone audio as WAV (16 kHz, 16-bit, mono) while transcribing.
- `Ctrl-C` / `Ctrl-D` interception for `watch` with a clear `"stopped"` JSON log line.
- `--auto-stop` and `--auto-stop-silence` for `mic` to stop when speech pauses.

## Change

- `watch` now runs indefinitely by default; stop it with `Ctrl-C` or `Ctrl-D`.

## Note

- `Ctrl-D` detection is only enabled when `stdin` is an interactive TTY, so piping into `watch` does not cause immediate exit.
