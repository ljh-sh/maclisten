## Add

- `maclisten locales` — list supported `SFSpeechRecognizer` locales.
- `maclisten file <path>` — transcribe an audio file.
- `maclisten mic` — record from the microphone and transcribe.
- `maclisten auth` — open System Settings privacy panes for Speech Recognition and Microphone.
- Hidden accessory `NSApplication` lifecycle so `Speech` framework callbacks work from a CLI binary.
- Compact JSON output for all commands.

## Note

- macOS TCC grants Speech Recognition / Microphone access to the terminal emulator app bundle (e.g. `Terminal.app`, `iTerm.app`, `Warp.app`), not to bare CLI processes. Run `maclisten` from a real terminal emulator.
