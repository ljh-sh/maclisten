## Add

- Region/language shortcuts on `file`, `mic`, and `watch`:
  - `--cn`, `--hk`, `--tw` → `zh-CN`, `zh-HK`, `zh-TW`
  - `--us`, `--gb`, `--au` → `en-US`, `en-GB`, `en-AU`
  - `--fr`, `--de`, `--es`, `--ja`, `--ko`, and many more → see `--help` for the full list
- Environment-aware default locale: if `--locale` / shortcuts are not given, read `$MACLISTEN_LOCALE`, then normalize `$LANG` (e.g. `de_DE.UTF-8` → `de-DE`), then fall back to `en-US`.
